FitYourGenes (HTML, CSS, Python, NodeJS, Bootstrap)

This web application uses Genome Link API and Flask to output genomic data. It utilizes genomic data to determine which diseases users are susceptible to, what nutritional deficiencies users have, and the body type of each user. The application returns information and personalizes foods, nutrients and exercises suggested for each individual. I competed with a group at UC San Diego’s hackathon for 36 hours, and was a front-end developer.

Spending Tracker (Python):

This program helps people set a budget and track their spending. If identified as a new user, a menu pops up, asking the person how much he or she wants to spend a day. If identified as a returning user, it already knows how much he or she wants to spend per day and it displays how much money the person has left. The user can open and close the application and keep inputting how much he or she spends that day and the program will constantly update itself. When it is a new day, the “money left to spend” will set itself back to the daily budget.
